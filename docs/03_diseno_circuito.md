# Diseño del Circuito (Pendiente)

## Esquema general
El circuito conecta:
- LED RGB a los pines D9, D10, D11.  
- Buzzer al pin D3.  
- Pulsador al pin D2 con resistencia pull-down.

## Diagrama (texto)
[D2] ← Pulsador
[D3] → Buzzer
[D9-D11] → LED RGB
[GND] → Común
[+5V] → Alimentación
