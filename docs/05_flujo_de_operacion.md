# Flujo de Operación (Pendiente)

1. Usuario presiona botón "Inicio".  
2. Arduino inicia temporizador → LED rojo.  
3. Al terminar el ciclo → buzzer + LED verde.  
4. Espera periodo de descanso.  
5. Repite ciclo o pausa manualmente.

## Diagrama
Usuario → Pulsador → Arduino → LED/Buzzer → Feedback
