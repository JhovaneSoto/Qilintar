# Flujo de Operación (Pendiente)

1. Arduino inicia temporizador → LED rojo.  
2. Al terminar el ciclo → buzzer + LED verde.  
3. Espera periodo de descanso.  
4. Repite ciclo o pausa manualmente.
5. Al completarse 4 ciclos normales, comienza un descanso largo.

## Diagrama
Usuario → Pulsador → Arduino → LED/Buzzer → Feedback
