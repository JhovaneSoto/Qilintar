# Firmware y Software

## Firmware (Arduino)
El firmware controla los ciclos Pomodoro:
1. Inicia en estado "trabajo" (LED rojo).  
2. Al completar el tiempo, pasa a "descanso" (LED verde).  
3. Suena el buzzer entre fases.  
4. Permite salto manual con pulsador.
