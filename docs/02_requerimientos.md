# Requerimientos

## Materiales y componentes
| Componente | Descripción | Cantidad |
|-------------|--------------|-----------|
| Arduino Nano | Microcontrolador principal | 1 |
| LED RGB | Indicador visual del estado | 1 |
| Buzzer | Alarma | 1 |
| Resistencias | Limitadoras de corriente  | 8 |
| Pulsador | Pausar/continuar o saltar fase | 3 |
| Protoboard y cables | Conexión temporal | Varios |

## Herramientas de software
- Visual Studio Code (Arduino Extension).

## Requerimientos funcionales
- Indicar con colores las fases.
  -  ![#FF3C00](https://placehold.co/15x15/FF3C00/FF3C00.png) Trabajo
  -  ![#00C878](https://placehold.co/15x15/00C878/00C878.png) Descanso
  -  ![#46A0FF](https://placehold.co/15x15/46A0FF/46A0FF.png) Descanso largo
- Activar buzzer al cambiar de fase.
- Pausar/continuar cada fase.
- Saltar fase.
